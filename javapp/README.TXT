This project abstract focuses on building multi purpose user friendly utility system,
where it has features such as QR code generator , password generator with checker,quote generator
,math quiz game,text to speak converter.the project aims is to create a user friendly fun experience when using the application.
so, main pillars of this application is HTML,CSS,JavaScript using these languages with API integration
,I have developed this system with sign up feature,ultimately contributing to a better understanding of a utility system.

APIs being used in the application:
- https://api.quotable.io/random
- https://api.qrserver.com/v1/create-qr-code
